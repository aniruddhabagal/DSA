# DSA (Data Structures and Algorithms)

A collection of classic algorithm implementations in C. This repository contains fundamental sorting and string matching algorithms commonly taught in computer science courses.

## Contents

- **Radix.c** - Radix sort implementation
- **insertion.c** - Insertion sort algorithm
- **kmp.c** - Knuth-Morris-Pratt string matching algorithm
- **patternmatch.c** - Basic pattern matching

## Building

To compile any of the C files:

```bash
gcc <filename>.c -o <filename>
```

For example:
```bash
gcc insertion.c -o insertion
```
