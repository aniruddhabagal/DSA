# DSA Repository

This is a Data Structures and Algorithms repository containing classic algorithm implementations in C. It includes fundamental sorting and string matching algorithms commonly taught in computer science courses.

## Project Structure

- **Radix.c** - Radix sort implementation
- **insertion.c** - Insertion sort algorithm
- **kmp.c** - Knuth-Morris-Pratt string matching algorithm
- **patternmatch.c** - Basic pattern matching

## Building and Running

To compile any of the C files, use GCC:

```bash
gcc <filename>.c -o <filename>
```

For example:
```bash
gcc insertion.c -o insertion
./insertion
```

## Getting Started for New Developers

1. Clone the repository
2. Choose an algorithm to explore (start with insertion.c for a simple sorting algorithm)
3. Compile and run the code to understand how it works
4. Modify the code to experiment with different inputs

No test framework is currently set up. To verify correctness, compare output with known correct results or reference implementations.
